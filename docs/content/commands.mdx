title: Commands
---

| Command             | Aliases | Sub-Command | Sub-Aliases | Usage                     | Description                                
| ---                 | ---     | ---         | ---         | ---                       | ---                                        |
| `/rhud`             | -       | -           | -           | `/rhud`                   | Toggle rHUD's visibility                   |
| `/rhud`             | -       | `config`    | -           | `/rhud config`            | Show rHUD's configuration panel            |
| `/rhud`             | -       | `reset`     | -           | `/rhud reset`             | Completely reset all config panel options  |
| `/postal <code>`    | -       | -           | -           | `/postal 001`             | Manage/create postal route                 |
| `/peacetime`        | `/pt`   | -           | -           | `/peacetime`              | Toggle peacetime mode                      |
| `/aop <text>`       | -       | -           | -           | `/aop Los Santos`         | Set AOP status                             |
| `/priority`         | `/pr`   | -           | -           | `/priority`               | Start a priority call                      |
| `/priority`         | `/pr`   | `cooldown`  | `cd`        | `/priority cooldown`      | Initiate priority cooldown                 |
| `/priority`         | `/pr`   | `onhold`    | `hold`      | `/priority onhold`        | Set priority on hold                       |
| `/priority`         | `/pr`   | `reset`     | `r`         | `/priority reset`         | Reset the priority timer                   |
| `/heal [player ID]` | -       | -           | -           | `/heal`, `/heal 1`        | Heal yourself or another player            |
