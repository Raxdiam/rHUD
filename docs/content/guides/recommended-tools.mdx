Below is a list of recommended tools that can be used to manage or edit your FiveM server and its files.

## Visual Studio Code
A free text editor that can be used to edit Lua, JavaScript, and other file types.

[Go to Download](https://code.visualstudio.com/)


### Recommended Extensions
- [Lua](https://marketplace.visualstudio.com/items?itemName=sumneko.lua): Provides Lua language support.
- [CfxLua (FiveM/RedM) IntelliSense](https://marketplace.visualstudio.com/items?itemName=overextended.cfxlua-vscode): Provides IntelliSense for FiveM and RedM Lua functions.
- [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode): Code formatter.
- [Remote - SSH](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-ssh): Connect to a remote server using SSH. Useful for opening a remote folder or file on your server directly in VSCode.

## WinSCP
A free SFTP, SCP, and FTP client for Windows that can be used to manage files on your server.

[Go to Download](https://winscp.net/eng/download.php)
